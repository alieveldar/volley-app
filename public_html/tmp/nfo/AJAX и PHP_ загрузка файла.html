AJAX и PHP: загрузка файла

Пример того как осуществить загрузку файла через PHP и jQuery ajax.

Для начала необходимо создать HTML элемент для загрузки файла.

view source
print?
1<input id="sortpicture" type="file" name="sortpic" />
2<button id="upload">Upload</button>

Далее необходимо дописать JS код:

view source
print?
01$('#upload').on('click', function() {
02    var file_data = $('#sortpicture').prop('files')[0];
03    var form_data = new FormData();
04    form_data.append('file', file_data);
05    alert(form_data);
06    $.ajax({
07                url: 'upload.php',
08                dataType: 'text',
09                cache: false,
10                contentType: false,
11                processData: false,
12                data: form_data,
13                type: 'post',
14                success: function(php_script_response){
15                    alert(php_script_response);
16                }
17     });
18});

Теперь приступим к PHP скрипту, где загрузим файл на сервер.

upload.php:

view source
print?
01<?php
02 
03    if ( 0 < $_FILES['file']['error'] ) {
04        echo 'Error: ' . $_FILES['file']['error'] . '<br>';
05    }
06    else {
07        move_uploaded_file($_FILES['file']['tmp_name'], 'uploads/' . $_FILES['file']['name']);
08    }
09 
10?>

Несколько слов о каталоге, в который загружается файл:

  1. Убедитесь, что путь к каталогу верен
  2. Каталог должен быть доступен для записи.

Пару слов о PHP функции move_uploaded_file, которая используется в upload.php:

view source
print?
1move_uploaded_file(
2 
3    // временное расположение файла
4    $_FILES['file']['tmp_name'],
5 
6    // конечный путь к файлу и его название
7    'uploads/' . $_FILES['file']['name']
8);

$_FILES['file']['name'] — изначальное имя файла. Можете задать своё название:

view source
print?
1move_uploaded_file(
2    $_FILES['file']['tmp_name'],
3    'uploads/my_new_filename.whatever'
4);

Также стоит убедиться что в конфигурационных настройках PHP, значения upload_max_filesize и post_max_size соответствуют вашим требованиям.
